earthdaily.legacy.APIRequester

class earthdaily.legacy.APIRequester(config, auth=None)[source]

Bases: object

Handles high-level API requests, managing authentication and service interactions.

This class is responsible for preparing API requests and delegating the actual HTTP calls to the HTTPClient. It also handles adding authorization headers.

Attributes:

The configuration object containing settings for the API requester.

The base URL for all API calls.

auth: Authentication

The authentication manager to obtain tokens for API requests.

http_client: HTTPClient

The HTTP client responsible for making actual HTTP requests.

headers: dict

The default headers to be included in every request.

send_request(request)[source]

Sends an HTTP request using the HTTPClient, adding authorization if necessary.

This method updates the ‘Authorization’ header of the request object with a ‘Bearer’ token obtained from the Authentication instance. It then sends the modified request through the HTTP client and returns the response.

Return type:

HTTPResponse
