earthdaily.Authentication

class earthdaily.Authentication(client_id, client_secret, token_url)[source]

Bases: object

Authentication class using the OAuth 2.0 client credentials flow.

This class exchanges client credentials (client_id and client_secret) for an access token at the token endpoint and handles token renewal when necessary.

Attributes:

client ID

client_secret: str

The token endpoint URL to retrieve the access token.

The current valid access token, initially None.

authenticate()[source]

Authenticates using client credentials and retrieves an access token.

This method initializes a session and configures it to use HTTP Basic Authentication with the provided client_id and client_secret. It performs an HTTP POST request to the token endpoint, specifically requesting an access token via the client credentials grant type.

Return type:

None

Raises:

ValueError:

If the authentication request fails, returns an invalid response, or if the response cannot be parsed as JSON.

get_token()[source]

Retrieves and returns a valid access token. If the current token is expired or not set, it re-authenticates to get a new token.
