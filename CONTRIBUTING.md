# Contributing to EarthDaily Python Client

Thank you for your interest in the EarthDaily Python Client!

## How to Contribute

This repository is maintained by the EarthDaily team. While we don't accept direct code contributions from external users, we greatly value your feedback and participation:

Found a bug? Please [create an issue](https://github.com/earthdaily/earthdaily-python-client/issues) with:
- A clear description of the problem
- Steps to reproduce the issue
- Expected vs actual behavior
- Your environment (OS, Python version, package version)

### üí° Feature Requests
Have an idea for a new feature? We'd love to hear it! [Open an issue](https://github.com/earthdaily/earthdaily-python-client/issues) describing:
- The feature you'd like to see
- Your use case and why it would be valuable
- Any examples or references

### üìñ Documentation Feedback
Found something unclear in our documentation? Let us know by [creating an issue](https://github.com/earthdaily/earthdaily-python-client/issues).
